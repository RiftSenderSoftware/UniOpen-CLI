# UniOpen-CLI

A lightweight Unity Editor tool that allows you to quickly open a CLI terminal directly from the Unity Project window.

## Features

- Open terminal in the current project directory.
- Fast access from the Unity Editor context menu.
- Supports daily development workflows.
- Lightweight and dependency-free.
- Easy installation through Unity Package Manager.

## Installation

### Via Unity Package Manager

Open Unity and navigate to:

Window → Package Manager

Click:

+ → Add package from Git URL...

Enter:

https://github.com/YourName/UniOpen-CLI.git

## Usage

1. Open the Project window.
2. Right-click on an empty area.
3. Select:

Open CLI

The terminal will open in the root directory of your Unity project.

## Supported Terminals

Depending on your operating system and configuration:

- Windows Terminal
- PowerShell
- Command Prompt
- Other custom CLI tools

## Why UniOpen-CLI?

Switching between Unity and a terminal is a common part of modern development workflows. UniOpen-CLI reduces friction by allowing developers to open a terminal directly from the Editor without manually navigating to the project folder.

Useful for:

- Git operations
- CLI-based AI coding assistants
- Build automation
- Package management
- Development scripts
