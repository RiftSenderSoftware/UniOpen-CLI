# Changelog

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [1.0.0] — 2026-06-12

### Added

- Контекстное меню **Assets → Open CLI** с запуском терминала в папке выбранного ассета.
- Окно настроек **Tools → UniOpenCLI**: добавление, редактирование и отключение команд.
- Кодогенерация пунктов меню под пользовательский список команд.
- Команды по умолчанию: Claude, Codex, Gemini (выключен).
- Пересборка PATH из реестра — CLI, установленные после запуска Unity, доступны без перезапуска редактора.
