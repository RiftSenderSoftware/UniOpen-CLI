# UniOpenCLI

Открывает терминал с нужным CLI-инструментом (Claude Code, Codex, Gemini CLI и любым другим) прямо из контекстного меню окна Project — в директории выбранного ассета.

## Возможности

- **ПКМ → Open CLI → ...** в окне Project — терминал откроется в папке выделенного ассета (для файла — в его папке, без выделения — в корне проекта).
- Настраиваемый список команд: имя, команда запуска, вкл/выкл.
- Пункты меню генерируются автоматически под ваш список команд.
- Настройки хранятся в `EditorPrefs` — индивидуальны для каждого разработчика и не засоряют репозиторий.
- PATH пересобирается из реестра при запуске — CLI, установленные после старта Unity, находятся без перезапуска редактора.

## Установка

Скопируйте папку `UniOpenCLI` в `Assets/Editor` вашего проекта. После компиляции пункты меню появятся автоматически.

Требования: Unity 2021.3+, Windows (терминал запускается через `powershell.exe`).

## Использование

1. Кликните правой кнопкой по любому ассету в окне Project.
2. Выберите **Open CLI → Claude** (или другую команду из списка).
3. Откроется PowerShell в директории ассета с уже запущенной командой.

## Настройка

Откройте **Tools → UniOpenCLI**:

- **Add CLI** — добавить команду (Name — текст пункта меню, Command — команда запуска).
- **Save** — сохранить и перегенерировать пункты меню.
- **Reset to Defaults** — вернуть список по умолчанию (Claude, Codex, Gemini).

Подробности — в [Documentation/Manual.md](Documentation/Manual.md).

## Структура пакета

```
UniOpenCLI/
├── UniOpenCLI.Editor.asmdef   — editor-only сборка
├── Scripts/                       — исходный код
│   ├── CliCommandData.cs          — сериализуемые модели команд
│   ├── CliCommandConfig.cs        — загрузка/сохранение настроек (EditorPrefs)
│   ├── CliContextMenu.cs          — запуск терминала в папке ассета
│   ├── CliMenuGenerator.cs        — кодогенерация пунктов меню
│   └── CliOpenerSettingsWindow.cs — окно настроек (Tools → UniOpenCLI)
├── Generated/                     — автогенерируемый код (не редактировать)
│   └── CliContextMenuGenerated.cs
├── Documentation/
│   └── Manual.md
├── README.md
└── CHANGELOG.md
```
